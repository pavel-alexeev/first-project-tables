HTML & CSS Cheatsheet

CSS Properties

Property Description
background-color Sets the background color of the element
border Specifies a border for an element
color Specifies the color of text in an element
display Specify an element's display behavior
font-family Sets the font family for an element
padding Sets the spacing between content and element border

HTML Tags

Tag Description
<h1> Represents a heading level 1
<input> Represents an input field
<nav> Represents navigation links
<span> Represents a section in a document
<figure> Represents a group of media content, and their caption
<embed> Represents external application or interactive content
